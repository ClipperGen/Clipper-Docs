# Clipper Docs

This repository contains the strategic proposals, research, business models, and technical research for the Clipper Gen platform.

## Structure

- **proposal/**: High-level strategy, business logic, technical workflow, and financial models.
- **riset/**: Foundational research, feasibility studies, and market trends.

This repository is integrated as a submodule in the main [Clipper Gen](https://github.com/ClipperGen/clipper-be) repository.
