## Bug Details & Resolusi (Untuk Laporan Perbaikan/Stabilitas)

### 1. FFmpeg Filtergraph Parsing (Filter not found)
* **The Problem (Masalah)**: Error `No such filter: '0'` atau `Filter not found` saat menjalankan pemotongan video portrait.
* **The Cause (Penyebab)**: Karakter koma dalam ekspresi `clip()` atau `min/max` pada filter crop disalahpahami oleh parser FFmpeg sebagai pemisah filter antar komponen.
* **The Solving (Solusi)**: Melakukan escaping pada karakter koma (`\,`) di dalam string ekspresi filter dan mengganti fungsi `clip` dengan kombinasi `min(max(...))` yang lebih robust.

### 2. NameError & Missing Dependencies
* **The Problem (Masalah)**: Worker mengalami crash dengan error `NameError: name 'logger' is not defined` dan kegagalan render karena font yang hilang.
* **The Cause (Penyebab)**: Modul `ffmpeg.py` belum mengimpor pustaka `logging`, dan font Roboto tidak tersedia di dalam image Docker Debian-based.
* **The Solving (Solusi)**: Menambahkan definisi logger standar, mengganti font default ke DejaVu Sans (yang tersedia secara sistem), dan menambahkan fallback untuk gaya "MrBeast".
