## Bug Details & Resolusi

### 1. "Stuck on Ingesting" / "Stuck on Analyzing"

* **The Problem**: Project video stuck on INGESTING or ANALYZING status, tidak pernah selesai. User report: "its stuck on ingestion and that just failed".

  - Beberapa video YouTube TIDAK MEMILIKI "Most Replayed" markers data
  - Video 6wysozn4e28: return 0 markers (No markers found in HTML)
  - Video dQw4w9WgXcQ (Rick Astley): return 99 markers ✓
  - Sistem sudah handle dengan fallback ke `_pick_fallback_anchors()`
  - Proses lanjut ke clip generation dengan predefined segments

* **The Solusi**: 
  - Heatmap function sudah robust (3 retries, multiple User-Agents, timeout handling)
  - Fallback logic sudah implement di `process_video_ingest` (lines 199-202)
  - Tidak perlu fix code - ini expected behavior
  - User need understand: Not all videos have heatmap data

### 2. CORS Errors (Non-Critical)

* **The Problem**: Browser console show CORS errors:
  - Cloudflare Insights beacon.min.js blocked
  - Font preload warnings (woff2 files)
  - Image proxy 400 errors for YouTube thumbnails

* **The Cause**: 
  - beacon.min.js: CSP blocking external script (not in script-src)
  - Fonts: Preload not used within timeframe
  - Thumbnails: next/image not configured for YouTube domains

* **The Solusi**:
  - [x] Disable Cloudflare Analytics in next.config.ts (Phase 1 complete)
  - [x] Add display: 'swap' to Geist fonts (Phase 1 complete)
  - [ ] Add remotePatterns for i.ytimg.com, ytimg.com (NEED FIX)
