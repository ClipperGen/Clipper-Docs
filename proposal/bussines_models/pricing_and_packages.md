## 2. Privileges Detail (ABAC Rules)

### **A. Free Tier (The Trial)**
*   Fokus pada aktivasi pengguna baru melalui **Welcome Bonus**.
*   Akses terbatas pada fitur "Magic Cut" berbasis data heatmap (berbayar token).
*   Output video wajib menyertakan watermark Clipper Gen.

### **B. Basic Creator (The Pro-Sumer)**
*   Didesain untuk YouTuber mingguan yang membutuhkan konsistensi tanpa watermark.
*   Mendapatkan alokasi token yang cukup untuk memproses ~5-7 jam video panjang per bulan.
*   Akses ke seluruh library subtitle dinamis (Hormozi-style, dsb).

### **C. Premium Agency (The Power House)**
*   Fitur kolaborasi tim (Multi-user) untuk manajemen akun klien.
*   **Deep Source Processing**: Kapasitas unggah file mentah (Direct Upload) hingga 5GB per file.
*   Dukungan teknis prioritas dan akses API untuk integrasi workflow internal agensi.
