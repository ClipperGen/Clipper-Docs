# Financial Strategy: CAPEX and OPEX Breakdown
**Capital Investment and Operational Sustainability Plan**

Analisis ini memisahkan pengeluaran modal awal (CAPEX) dari biaya operasional berkelanjutan (OPEX) untuk memastikan efisiensi arus kas selama 6 bulan pertama hingga tahap skalabilitas.

## 1. Capital Expenditure (CAPEX)
CAPEX mencakup investasi satu kali untuk aset dan fondasi legal yang diperlukan sebelum platform beroperasi penuh.

| Komponen Investasi | Deskripsi | Estimasi Biaya |
| :--- | :--- | :--- |
| **Legal & Incorporation** | Pendirian PT/CV, Akta, dan Izin Usaha. | Rp 15.000.000 |
| **Brand Identity** | Desain Logo, UI Kit Pro, dan Aset Pemasaran Awal. | Rp 10.000.000 |
| **Development Hardware** | Pengadaan Workstation Core Team (2 Unit High-end). | Rp 35.000.000 |
| **Initial R&D (MVP)** | Biaya pengembangan fitur inti hingga Alpha launch. | Rp 120.000.000 |
| **SaaS Lifetime Licenses** | Lisensi software/font komersial untuk kebutuhan desain. | Rp 5.000.000 |
| **Total CAPEX** | | **Rp 185.000.000** |

## 2. Operating Expenditure (OPEX)
OPEX mencakup biaya bulanan yang diperlukan untuk menjalankan layanan dan melayani pengguna.

| Biaya Operasional | Estimasi Bulanan | Total (6 Bulan) |
| :--- | :--- | :--- |
| **Cloud GPU (AWS/GCP)** | Biaya server pemrosesan video & worker. | Rp 10.000.000 | Rp 60.000.000 |
| **AI API Fees (GPT-4o)** | Biaya inferensi semantik & refinement. | Rp 2.000.000 | Rp 12.000.000 |
| **Maintenance Salaries** | Gaji tim engineering & support pasca-MVP. | Rp 15.000.000 | Rp 90.000.000 |
| **Digital Marketing** | Biaya iklan (Ads) & Akuisisi User. | Rp 5.000.000 | Rp 30.000.000 |
| **SaaS & Cloud Storage** | S3, Redis, PostgreSQL, Clerk, Monitoring. | Rp 2.500.000 | Rp 15.000.000 |
| **Total OPEX (6 Mo)** | | **Rp 207.000.000** |

## 3. Financial Summary & Contingency
Total pengeluaran dikonsolidasikan dengan penambahan cadangan risiko.

*   **Total Base Cost (CAPEX + OPEX)**: Rp 392.000.000
*   **Contingency Fund (15%)**: Rp 58.800.000
*   **Total Budget Required**: **Rp 450.800.000**

## 4. Sustainability Strategy
Untuk menjaga efisiensi OPEX, Clipper Gen menerapkan:
1.  **Embedding-First Policy**: Mengurangi beban biaya API LLM hingga 80%.
2.  **Spot-Instance Workers**: Menggunakan GPU cadangan dengan harga 60% lebih murah untuk proses non-priority.
3.  **Growth-Driven Scaling**: Infrastruktur hanya akan melakukan *scaling-up* secara otomatis seiring dengan pertumbuhan pendapatan dari Top-up token.
