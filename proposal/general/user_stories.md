# User Stories: Clipper Gen Experience
**Defining the Journey for Every User Role**

Dokumen ini mendefinisikan kebutuhan fungsional platform melalui sudut pandang pengguna akhir (User Stories).

## 1. Visitor / Unauthenticated User
*   **Discovery**: Sebagai pengunjung, saya ingin melihat demo hasil klip AI agar saya yakin dengan kualitas platform sebelum mendaftar.
*   **Pricing**: Sebagai pengunjung, saya ingin melihat tabel perbandingan paket agar saya bisa memilih paket yang sesuai dengan budget saya.

## 2. Free User
*   **YouTube Ingest**: Sebagai pengguna gratis, saya ingin memasukkan link YouTube agar saya bisa mencoba fitur auto-clip secara instan.
*   **One-time Welcome Bonus**: Sebagai pengguna baru, saya ingin mendapatkan 50 token gratis setelah verifikasi email agar saya bisa langsung mencoba membuat klip pertama saya.
*   **Referral Earning**: Sebagai pengguna gratis, saya ingin membagikan link referral saya kepada teman-teman sesama kreator agar saya bisa mendapatkan tambahan token gratis saat mereka bergabung.
*   **Magic Cut**: Sebagai pengguna gratis, saya ingin AI memindai video saya dan memberikan klip pendek dengan watermark agar saya bisa melihat hasil kerjanya sebelum memutuskan berlangganan.

## 3. Basic & Premium User
*   **Direct Upload (Premium Only)**: Sebagai pengguna Premium, saya ingin mengunggah file mentah dari PC saya agar saya bisa memproses konten yang belum dipublikasikan ke YouTube.
*   **No Watermark**: Sebagai pengguna berbayar, saya ingin mengekspor klip tanpa watermark agar konten saya terlihat profesional.
*   **Advanced Editing**: Sebagai kreator, saya ingin mengedit teks subtitle dan gaya animasi agar sesuai dengan branding unik saya.
*   **Top-up Tokens**: Sebagai agensi, saya ingin membeli token tambahan saat token bonus saya habis agar proyek klien saya tidak terhambat.

## 4. Admin / Developer
*   **GPU Monitoring**: Sebagai pengembang, saya ingin melihat status beban GPU agar saya bisa melakukan scaling server jika antrean terlalu panjang.
*   **Token Management**: Sebagai admin, saya ingin bisa memberikan token kompensasi kepada user jika terjadi kegagalan sistem pada proses rendering.
